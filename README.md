# Привет! Я Савелий | Manual QA

> Изучал Frontend разработку, но понял, что поиск дефектов и обеспечение качества ПО мне ближе. Знания (HTML, CSS, JS) помогают мне глубже понимать веб-приложения.


### Навыки

- **Тест-анализ и тест-дизайн**
  - Написание тест-кейсов, чек-листов
  - Применение техник тест-дизайна (классы эквивалентности, граничные значения, use case)
- **Виды тестирования**
  - Функциональное и нефункциональное тестирование
  - Тестирование UI/UX и верстки (кроссбраузерность, адаптивность)
  - Позитивное и негивное тестирование
  - Регрессионное, smoke, sanity
- **Документирование**
  - Создание понятных и полных баг-репортов
  - Ведение тестовой документации
- **Тестирование API**
  - Ручное тестирование API через Postman
